MANUAL_TÉCNICO

🎮 [BOOTCAMP: PRIMEROS PASOS]

⚡ CONSOLE.LOG (Tu Megáfono)

Muestra mensajes en la terminal. Es lo primero que vas a aprender. ¡Así te comunicás con la estación!

console.log('Hola Estación!');
// Esto aparece en la terminal de abajo

📦 VARIABLES (Cajas de Datos)

Guardá información para usarla después. Usá let para valores que cambian y const para los que no.

let energia = 100; // puede cambiar
const NOMBRE = 'Hércules'; // no cambia

🔢 NÚMEROS vs TEXTOS

Los números van sin comillas. Los textos (strings) van SIEMPRE entre comillas.

let nivel = 5; // número
let nombre = 'STARK'; // texto
console.log(nombre + ' nivel ' + nivel);

❓ IF / ELSE (Decisiones)

Como un guardia: SI se cumple la condición, hacé algo. SI NO, hacé otra cosa.

if (energia > 50) {
  console.log('SISTEMA_OK');
} else {
  console.log('¡ALERTA!');
}

🤖 FUNCIONES (Robots)

Un bloque de código con nombre. Lo creás una vez y lo usás las veces que quieras.

function saludar() {
  console.log('¡HOLA!');
}
saludar(); // la ejecutás así

📋 ARRAYS (Listas)

Una lista ordenada. Se crea con corchetes. ¡OJO: empiezan en posición 0!

let armas = ['Láser', 'Misil', 'Escudo'];
console.log(armas[0]); // 'Láser'
console.log(armas.length); // 3

🔄 BUCLE FOR (Repetir)

Repetí código automáticamente. Perfecto para recorrer listas.

for (let i = 0; i < 3; i++) {
  console.log('Escaneo ' + i);
}
🖥️ [INTERACCIÓN CON LA PANTALLA]

🔍 getElementById (Buscar Elementos)

Buscá cualquier elemento de la pantalla por su nombre (ID) para modificarlo.

const visor = document.getElementById('sarge-text');

✏️ innerText / innerHTML (Cambiar Texto)

Cambiá lo que dice un elemento en la pantalla.

visor.innerText = 'NUEVO MENSAJE';
visor.innerHTML = '<b>CON FORMATO</b>';

🎨 .style (Cambiar Apariencia)

Modificá colores, tamaños, bordes, y más desde el código.

elemento.style.borderColor = 'red';
elemento.style.opacity = '0.5';
elemento.style.width = '200px';

👆 addEventListener (Detectar Clicks)

Hacé que algo pase cuando el usuario toca un botón.

boton.addEventListener('click', () => {
  console.log('¡CLICK!');
});

➕ createElement + appendChild (Crear Elementos)

Creá nuevos elementos HTML y agregalos a la página.

const div = document.createElement('div');
div.innerText = 'ALERTA';
document.getElementById('terminal').appendChild(div);
⚙️ [JS INTERMEDIO: MÉTODOS Y DATOS]

🧹 .filter() (Filtrar Listas)

Creá una nueva lista quedándote solo con los elementos que cumplan una condición.

const criticos = logs.filter(l => l.critico === true);

🔄 .map() (Transformar Listas)

Creá una nueva lista transformando cada elemento.

const dobles = numeros.map(n => n * 2);

📦 Desestructuración (Extraer Datos)

Sacá propiedades de un objeto directamente a variables.

const { lat, long } = coordenadas;

🔀 Switch (Múltiples Opciones)

Como varios if/else juntos. Ideal para menús o estados.

switch(alerta) {
  case 'ALTA': activar(); break;
  case 'BAJA': reposo(); break;
}

🌐 Fetch (Pedir Datos)

Conectate a servidores y pedí información.

fetch('https://api.datos.com')
  .then(r => r.json())
  .catch(e => console.error(e));

💾 localStorage (Guardar Datos)

Guardá datos que sobrevivan al cerrar la página.

localStorage.setItem('score', '100');
const score = localStorage.getItem('score');
🛡️ [TYPESCRIPT: BLINDAJE DE DATOS]

🏷️ Tipos Básicos

TypeScript le pone "etiquetas" a tus datos para evitar errores.

let id: string = 'SEC-01';
let nivel: number = 100;
let activo: boolean = true;

📐 Interface (Planos)

Define la forma EXACTA que debe tener un objeto.

interface Piloto {
  nombre: string;
  rango: number;
}

🔒 Readonly (Inmutable)

Marcá propiedades que NADIE puede cambiar después de crearlas.

interface Config {
  readonly id: string;
}

📊 Enum (Opciones Fijas)

Una lista cerrada de valores posibles. Perfecto para estados.

enum Alerta { BAJA, MEDIA, CRITICA }
let estado = Alerta.CRITICA;

🔗 Union (|) e Intersección (&)

Union: puede ser UNA cosa u OTRA. Intersección: tiene TODO de ambos.

type Resp = string | number; // una u otra
type Full = Base & Extra; // tiene ambas
🧬 [TS AVANZADO: HERRAMIENTAS PRO]

🎁 Genéricos <T> (Comodines)

Creá funciones e interfaces que trabajen con CUALQUIER tipo.

interface Caja<T> {
  contenido: T;
}
// Caja<string>, Caja<number>...

🔎 typeof (Detective de Tipos)

Preguntá qué tipo tiene una variable en tiempo real.

if (typeof valor === 'number') {
  // Acá TS sabe que es número
}

🧩 Partial, Pick, Omit, Record

Herramientas para modificar interfaces rápidamente.

Partial<T> // todo opcional
Pick<T, 'a' | 'b'> // solo estas props
Omit<T, 'pass'> // sin esta prop
Record<string, number> // diccionario

❄️ as const (Congelar)

Hace que un objeto sea 100% inmutable. Más potente que readonly.

const CONFIG = { v: 1 } as const;

⚡ Async / Await + Tipos

Funciones asíncronas con tipos estrictos para mayor seguridad.

async function cargar(): Promise<string> {
  return 'LISTO';
}
